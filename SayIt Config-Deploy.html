Multiple Versions / Subdirs
One APK serves all versions in one repo. Switch the active subdir inside the app (home.html gear icon → change subdir). For a different repo, run this tool with new config and install a separate APK.
Verification Checklist
Run through these after deploying.
Common Issues
404 on GH Pages URL — Pages must be enabled. Wait 3 min after first push.
Voice not working — Grant Microphone to the APK: App Settings → Permissions → Microphone.
PAT rejected — Regenerate with Contents read+write. Check it has not expired.
AI platform won't open — Must be logged in inside the APK browser.
Blank screen in APK — Verify GH Pages is live first. Java has the correct URL from your config.
PAT expired — Open app → gear icon → Update GitHub Token → enter new token. No files to regenerate, no GitHub upload, no reinstall.
New version needed — Bump Version Number on step 1, re-run step 2. Switch subdir in app settings.
