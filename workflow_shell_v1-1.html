System Strategy

Native belongs in the helper. Flexibility belongs in the web shell. The helper is a configurable capability bridge for clipboard, intents, URL launch, storage, logging, optional location, files, notifications, and voice. The web shell owns workflow logic, configuration, routing, module registration, helper app registration, and lifecycle orchestration.
